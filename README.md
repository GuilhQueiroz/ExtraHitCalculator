# ExtraHit
Aplicativo simples para calcular automaticamente quantos golpes extras meu personagem pode fazer em um RPG de mesa.

## **Como funciona?**
* Informe os 5 ataques de dano. O app calculará quantos ataques extras você pode fazer.
